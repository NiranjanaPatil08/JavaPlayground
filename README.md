# Java Playground ☕

A personal Java repository created for revising, practicing, and strengthening Core Java concepts.

This repository contains small programs, examples, and implementations while revisiting Java fundamentals and advanced concepts.

The goal is to build a strong Java foundation before developing production-level backend applications using Spring Boot.

---

# Purpose

This repository is created to:

- Refresh Core Java fundamentals
- Practice Object-Oriented Programming concepts
- Understand Java APIs and best practices
- Experiment with Java features through hands-on coding
- Prepare for Spring Boot and backend development
- Maintain notes and examples for future reference
